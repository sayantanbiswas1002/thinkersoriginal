Form tag:

This is the tag used to initiate any form and is paired.

Input Tag:
It is perhaps the most used tag in forms. It is a tag that takes input of a certain data.
It has got various attributes like type, name etc.

'Type' attribute of input tag:

 It mentions the type of input.
Some of the types are:
  • Radio Type
  • Text type
  • Checkbox type

Action:

The action attribute defines the action to be performed when the form is submitted. Normally, the form data is sent to a web page on the server when the user clicks on the submit button.

Target:

The target attribute specifies if the submitted result will open in a new browser tab, a frame, or in the current window. The default value is "_self" which means the form will be submitted in the current window.

Method:

The method attribute specifies the HTTP method (GET or POST) to be used when submitting the form data.The default method when submitting form data is GET. However, when GET is used, the submitted form data will be visible in the page address field. Post is used when you want data to be remove data after submitting
